# PISA2018-Global-Competence-Achievement

## Research Purpose
This study examines the relationship between global competence and academic achievement using PISA 2018 data.

## Data Source
Programme for International Student Assessment (PISA) 2018, published by the OECD.

## Countries
31 participating countries.

## Country Grouping Criteria

### Development Level Classification
Countries were grouped according to the economic development classification provided by the International Monetary Fund (IMF), based on the World Economic Outlook database (2022 edition).

Source:
https://www.imf.org/external/pubs/ft/weo/2022/01/weodata/groups.htm

### Cultural Value Classification
Cultural value groupings were constructed using data from the World Values Survey (WVS), based on aggregated national-level cultural indicators.

Source:
https://www.worldvaluessurvey.org/WVSDocumentationWV7.jsp

## Variables

### Independent Variables
- GCSELFEFF
- RESPECT
- AWACOM
- GLOBMIND

### Dependent Variables
- MATH (PV1–PV10)
- READ (PV1–PV10)
- SCIE (PV1–PV10)

### Control Variables
- ST004D01T
- ST013Q01TA
- IMMIG
- ESCS
- SC013Q01TA 

## Methodology
- Multiple Imputation (Fully Conditional Specification, Predictive Mean Matching)
- Ordinary Least Squares (OLS) Regression
- Grouped Regression Analysis (by development level and cultural value clusters)

## Software
IBM SPSS Statistics 27

## Data Availability
PISA 2018 data can be obtained from the OECD official website:
https://www.oecd.org/pisa/data/

IMF development level classification can be accessed from:
https://www.imf.org/external/pubs/ft/weo/2022/01/weodata/groups.htm

World Values Survey data can be accessed from:
https://www.worldvaluessurvey.org/WVSDocumentationWV7.jsp
